Windows 11 / Edge Troubleshooting
  1. If the popup says "Gekoppelt" (Paired) next to the droid, the connection will fail.
  2. Right-click your Windows Start button → Device Manager.
  3. Click View → Show hidden devices.
  4. Expand the Bluetooth category. Find D2-XXXX, right-click it, and select Uninstall device.
  5. Turn the droid off, then turn it back on.
  6. Refresh this webpage and try connecting again.
