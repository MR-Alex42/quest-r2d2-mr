Connection Troubleshooting
  1. Quest 3: Ensure the droid is NOT paired in the main Quest OS Bluetooth settings. If it says "Previously Connected", forget/unpair it.
  2. Windows 11: Open Device Manager → View → Show hidden devices. Look under Bluetooth, uninstall D2-XXXX.
  3. Turn the droid completely off, wait 3 seconds, turn it back on.
  4. Refresh this webpage and try connecting again.
